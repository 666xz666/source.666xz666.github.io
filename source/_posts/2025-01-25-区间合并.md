# 区间合并

## 主要内容

快速地将一系列区间中有交集（包括端点相交）的区间合并成一个,返回处理后的区间列表。

![image-20250125163700640](/images/posts/image-20250125163700640.png)

## 代码模板

```cpp
void merge(vector<PII>& segs){
    vector<PII> res;

    sort(segs.begin(),segs.end());//先按照左边界大小从小到大排序

    int st=-2e9,ed=-2e9;
    for(auto seg:segs){
        if(ed<seg.first){
            if(ed!=-2e9) res.push_back({st,ed});
            st=seg.first,ed=seg.second;
        }
        else ed=max(ed,seg.second);
    }

    if(ed!=-2e9) res.push_back({st,ed});//要记得把最后一段区间加到结果里

    segs=res;
}
```

typedef pair<int,int> PII;
vector<PII> segs;
int n;

void merge(vector<PII>& segs){
    vector<PII> res;

    sort(segs.begin(),segs.end());//先按照左边界大小从小到大排序

    int st=-2e9,ed=-2e9;
    for(auto seg:segs){
        if(ed<seg.first){
            if(ed!=-2e9) res.push_back({st,ed});
            st=seg.first,ed=seg.second;
        }
        else ed=max(ed,seg.second);
    }

    if(ed!=-2e9) res.push_back({st,ed});//要记得把最后一段区间加到结果里

    segs=res;
}

void work(){
    cin>>n;
