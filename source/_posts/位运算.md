# 位运算

## 主要内容

### low_bit

返回的是一个整数的二进制形式最右边的“1”，比如对于(101011000)<sub>2</sub>

返回的是(1000)<sub>2</sub>

```cpp
int low_bit(int n){
    return n&-n;
}
```

## 例题

### low_bit

二进制表示中1的个数

```cpp
#include<bits/stdc++.h>
using namespace std;

int low_bit(int n){
    /*
    可以得到整数二进制形式的最右边的'1'
    比如(101101000)2
    得到(1000)2 也就是8
    */
    return n&-n;
}

void work(){
    int n;
    cin>>n;

    int cnt=0;
    while(n) cnt++,n-=low_bit(n); 
    cout<<cnt<<endl;
}

int main(){
    work();
    return 0;
}
```
